## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## How to Use

1. Select the input format (auto-detected from uploaded file)
2. Select the desired output format
3. Drag and drop an image or click to browse
4. Click "Convert" to process your image
5. Download your converted image
